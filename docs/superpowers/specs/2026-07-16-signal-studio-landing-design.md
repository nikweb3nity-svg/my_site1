# «Сигнал» — лендинг студии веб-дизайна

## Цель

Создать одностраничный русскоязычный сайт вымышленной студии «Сигнал», которая проектирует сайты и цифровые айдентики. Лендинг должен буквально опираться на переданную дизайн-систему: чёрный canvas, крупная лёгкая типографика, единый фиолетовый CTA и генеративная анимированная графика частиц. Название, тексты, логотип и иллюстрация — оригинальные.

## Структура

1. Прозрачная шапка: знак, «Сигнал», якоря «Работы», «Подход», «Команда», CTA «Обсудить проект».
2. Hero: крупный слоган слева и интерактивное облако цветных треугольных частиц справа.
3. Работы: три отобранных проекта как свободные, крупные типографические композиции без карточек.
4. Подход: асимметричная двухколоночная секция с коротким манифестом и анимированной схемой процесса.
5. Команда: минимальные профили без рамок и теней.
6. Контактный финал с единственным фиолетовым CTA.

## Визуальная система

- Фон всех секций: `#000000`; исключены панели, тени и видимые разделители.
- Основной текст: белый; вторичный: `#9a9a9a` и `#bdbdbd`.
- Единый насыщенный интерактивный акцент: `#8052ff`; янтарный `#ffb829` используется только для подписей и акцентов.
- Заголовки: regular 400, крупный масштаб, отрицательный трекинг. Основной текст: weight 200.
- Макет: максимум 1280px, просторные двухколоночные композиции и интервалы 60–120px.
- Кнопки: фиолетовые пилюли с радиусом 24px. Вторичные действия — текстовые ссылки.

## Анимация и взаимодействия

- Canvas-облако состоит из мелких цветных треугольников, медленно дрейфует и заметно реагирует на курсор.
- Небольшие фоновые частицы продолжают движение вокруг ключевых секций.
- Секции и строки заголовков мягко проявляются при прокрутке; движения используют opacity/transform, учитывают `prefers-reduced-motion`.
- Якоря навигации и CTA ведут к существующим секциям; мобильное меню раскрывается без сторонних библиотек.

## Адаптивность и доступность

- На узких экранах колонки становятся одной, визуализация остаётся между hero-текстом и CTA.
- Семантические секции, видимый focus, достаточный контраст, кнопки и ссылки доступны с клавиатуры.
- При `prefers-reduced-motion: reduce` движение частиц и reveal-анимации отключаются.

## Техническая реализация

Лендинг — статический `index.html` с локальными `styles.css` и `script.js`. Переданные CSS-токены перенесены в `styles.css`; canvas API используется для оригинальной процедурной графики. Внешние зависимости не требуются.
